# My Personal Portfolio 🚀

Welcome to my first web development project! This is a clean and responsive portfolio website designed to showcase my skills and journey as a computer science student.

## 🛠️ Tech Stack
- **HTML5**: For structured and semantic web content.
- **CSS3**: For custom styling, layouts, and responsive design.

- **Skills Section**: Highlighting my core programming competencies.
- **Projects**: A display of my latest work and coding progress.
